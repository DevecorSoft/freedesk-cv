# Freedesk-cv

the cv component of [freedesk](https://github.com/freedesk)

## Setup

### Install opencv with python

```shell
python -m pip install opencv-python
```

### Load opencv data

```shell
cd freedesk-cv
git clone git@github.com:opencv/opencv.git
ln -s opencv/data data
```

### Run

```shell
python main.py
```
